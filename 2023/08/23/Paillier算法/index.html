Paillier算法属于半同态加密算法

什么是同态加密(HE)？

同态加密是一种特殊的加密方法，它允许直接对加密数据执行计算，如加法和乘法，而计算过程不会泄露原文的任何信息。计算的结果仍然是加密的，拥有密钥的用户对处理过的密文数据进行解密后，得到的正好是处理后原文的结果。

根据支持的计算类型和支持程度，同态加密可以分为以下三种类型：半同态加密，部分同态加密，全同态加密

半同态加密

半同态加密(Partially Homomorphic Encryption, PHE)：只支持加法或乘法中的一种运算。其中，只支持加法运算的又叫加法同态加密（Additive Homomorphic Encryption, AHE），常见的有RSA，Elgamal，Paillier

部分同态加密

部分同态加密(Somewhat Homomorphic Encryption, SWHE)：可同时支持加法和乘法运算，但支持的计算次数有限

全同态加密

全同态加密(Fully Homomorphic Encryption, FHE)：支持任意次的加法和乘法运算

第一个满足加法和乘法同态的同态加密方法直到2009年才由Craig Gentry提出。目前来说，全同态加密算法性能较差，应用较少。比较常用的是半同态加密算法。

Paillier算法

密钥生成

  1. 随机选两个大素数$p,q$，满足$gcd(pq,(p-1)(q-1)) = 1$，且$p,q$长度相同（安全性高一点）

  2. 计算$n = pq$，以及$\lambda = lcm(p-1,q-1)$，即$\lambda$是$p-1$和$q-1$的最小公倍数

  3. 随机选择$g \in Z_{n^2}^*$，表示$g$为$(0,n^2)$中的正整数，g还需满足$gcd(L(g^{\lambda} \mod n^2),n) = 1$。后面发现$g = n+1$满足条件，可以直接取$g = n+1$

  4. 定义$L$函数：$L(x) = \frac{x-1}{n}$，计算$\mu = (L(g^{\lambda}\mod n^2))^{-1} \mod n$

  5. 公钥为$(n,g)$，私钥为$(\lambda,\mu)$

加密

  1. 明文$m$，满足$m \in Z_n$

  2. 选择随机数$r$，满足$r \in Z_n^*$

  3. 计算密文$c = g^mr^n \mod n^2$

解密

$m = L(c^\lambda \mod n^2)×\mu \mod n$即$m = \frac{L(c^\lambda \mod n^2)}{L(g^\lambda \mod n^2)} \mod n$

解密的正确性

根据二项式定理

$\because g = n+1,\therefore L(g^\lambda \mod n^2) = \lambda$

$\therefore \mu = (L(g^\lambda \mod n^2))^{-1} \mod n = \lambda^{-1} \mod n$

$m=L(c^\lambda \mod n^2)×\mu \mod n= L((g^mr^n)^\lambda \mod n^2)× \lambda^{-1} \mod n$

又$\because L((g^mr^n)^\lambda \mod n^2) = L(g^{\lambda m}r^{\lambda n} \mod n^2)$

而这个$\lambda n = \frac{(p-1)(q-1)}{k}×pq$，其实就是$\frac{\phi(n^2)}{k}$

$\therefore L((g^mr^n)^\lambda \mod n^2) = L(g^{\lambda m}r^{\lambda n} \mod n^2) = \lambda m \mod n^2$

$\therefore m=L(c^\lambda \mod n^2)×\mu \mod n= L((g^mr^n)^\lambda \mod n^2)× \lambda^{-1} \mod n = \lambda m ×\lambda^{-1} \mod n = m$

同态加法的性质

对于任意$m_1,m_2 \in Z_n$和任意$r_1,r_2 \in Z_n^*$

对应密文$c_1 = E(m_1,r_1) = g^{m_1}r_1^n$，$c_2 = E(m_2,r_2) = g^{m_2}r_2^{n}$

把密文作乘法$c = c_1×c_2 = g^{m_1+m_2}(r_1×r_2)^n \mod n^2$

把密文c解密得到的是$m_1+m_2$

即我们得到了：$c_1 × c_2 = m_1 + m_2$，即密文乘等于明文加。
